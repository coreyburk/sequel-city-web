# History Endpoints

The history contract exposes backend-owned query execution records. It is currently process-local and in-memory only.

## `GET /api/query/history`

Returns the current in-memory query history list.

### Response Codes

| Status | When returned |
|---|---|
| `200` | History loaded successfully |
| `500` | History loading failed |

### Success Shape

| Field | Type | Notes |
|---|---|---|
| `success` | `true` | Always `true` on success |
| `data.records` | `QueryHistoryRecord[]` | Records returned in reverse chronological order |

### `QueryHistoryRecord`

| Field | Type | Notes |
|---|---|---|
| `id` | `number` | Backend-generated sequential id |
| `timestamp` | `string` | ISO 8601 timestamp generated by the backend |
| `queryText` | `string` | Original submitted SQL text |
| `outcome` | `"success" \| "blocked" \| "failed"` | Backend-owned outcome classification |
| `rowCount` | `number \| null` | Present for successful query execution |
| `executionTimeMs` | `number \| null` | Recorded backend execution time |
| `errorMessage` | `string \| null` | Safety or execution failure detail when available |

### Example `200`

```json
{
  "success": true,
  "data": {
    "records": [
      {
        "id": 2,
        "timestamp": "2026-05-05T10:00:01.000Z",
        "queryText": "SELECT 2",
        "outcome": "success",
        "rowCount": 1,
        "executionTimeMs": 11,
        "errorMessage": null
      },
      {
        "id": 1,
        "timestamp": "2026-05-05T10:00:00.000Z",
        "queryText": "DELETE FROM PersonsOfInterest",
        "outcome": "blocked",
        "rowCount": null,
        "executionTimeMs": 2,
        "errorMessage": "DELETE statements are not allowed."
      }
    ]
  }
}
```

### Example `500`

```json
{
  "success": false,
  "message": "Unable to load query history."
}
```

## Contract Notes

- Records are not persisted across backend restarts.
- The endpoint does not currently support pagination, filtering, or deletion.
- The history service records blocked, failed, and successful execution attempts.
