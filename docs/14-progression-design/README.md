# Progression Design Package

This package documents Sequel City Web Detective's future-facing detective rank, reward, badge, and case-tier system.

It is a design package only. It does not describe already implemented runtime behavior unless explicitly stated.

## Contents

- `Detective-Rank-and-Reward-System-Guide.md`
  The official working guide for career tiers, progression rules, reward terminology, badge intent, UI language standards, and current case placement guidance.

## Package Purpose

Use this package when work needs to answer questions such as:

- What rank ladder should students progress through?
- How should case completion contribute to long-term advancement?
- What should replace or refine current reward labels?
- How should future cases be tiered?
- What language should the UI use for rank, badges, and rewards?

## Boundary

- This package may guide future UI and progression implementation work.
- It does not override the current runtime unless a later accepted implementation work package applies it in code.
